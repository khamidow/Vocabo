## ✨ Features

| Feature | Description |
|----------|-------------|
| 📖 Word Learning | Learn new English words with meanings and examples |
| 🧠 Smart Practice | Practice vocabulary using interactive exercises |
| 🎤 STT Search | Find translations easily just by asking |
| ⭐ Save Words | Bookmark important words for later revision |
| 🔍 Search Dictionary | Quickly find word definitions |
| 🌙 Clean UI | Simple and modern Dark/Light UI for any device |

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Components
- Room Database
- MVVM Architecture
- Coroutines & Flow
- ViewBinding

---

## 🚀 Installation

```bash
git clone https://github.com/khamidow/Vocabo.git
```

```bash
cd Vocabo
```

Then open the project in Android Studio and run it on an emulator or physical device.

---

## 📌 About

Vocabo is a lightweight vocabulary learning application designed to help users improve their English word knowledge through daily practice, quizzes, and structured learning.
